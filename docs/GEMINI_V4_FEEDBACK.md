## Shadow Mode란?

```
1. 봇은 매일 밤 실제로 돈다
2. 데이터는 실제로 쌓인다
3. 단, '등록' 버튼만 비활성화
4. 1주일간 사장님이 직접 검수
5. 튜닝 완료 후 실전 투입
```
