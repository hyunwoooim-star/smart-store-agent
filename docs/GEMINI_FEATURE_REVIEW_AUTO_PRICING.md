## 1. 현재 Pain Points (사용자 피드백)

### 문제 1: 1688 사이트 정보 파악 어려움
```
- 중국어라 어디가 MOQ인지, 무게인지, 사이즈인지 모름
- 로그인 팝업이 가려서 정보 안 보임
- 수동으로 하나하나 찾아서 복사해야 함
```

### 문제 2: 목표가 설정 막막함
```
- "목표가를 얼마로 잡아야 하지?"

- 어떤 가격이 적절한지 판단 어려움
```

### 문제 3: 전체 프로세스가 수동
```
현재: 1688 확인 → 수동 입력 → 네이버 검색 → 목표가 입력 → 마진 분석
시간: 상품 1개당 10-15분 소요
```

## 2. 제안 기능: 원클릭 마진 분석

### 이상적인 UX Flow

```
[STEP 1] 사용자: 1688 스크린샷 붙여넣기 (또는 URL)
         ↓
[STEP 2] AI: 상품 정보 자동 추출
         - 상품명 (번역)
         - 가격 (¥)
         - 무게 (kg)
         - 사이즈 (cm)
         - MOQ
         ↓
[STEP 3] AI: 한국 마켓플레이스 자동 검색
         - 네이버 쇼핑
         - 쿠팡
         - 유사 상품 가격 수집
         ↓
[STEP 4] AI: 목표가 자동 추천
         - 시장 최저가: 38,500원
         - 시장 평균가: 41,200원
         - 추천 목표가: 38,400원 (최저가 - 100원)
         - 근거 링크 제시
         ↓
[STEP 5] AI: 마진 분석 자동 실행
         ↓
[STEP 6] 결과: "🟢 진입 추천" or "🔴 진입 금지"
```
